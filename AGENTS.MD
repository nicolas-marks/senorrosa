# Señor Rosa Development Guide

## Mission

Señor Rosa is not a game.

It is a storytelling platform built around illustrated true stories from real life.

Every decision should make it easier to tell, publish, and enjoy authentic stories.

---

# Core Principles

1. Story comes before technology.
2. Reading should be effortless.
3. Mobile experience is the priority.
4. Simplicity beats cleverness.
5. Every page should load quickly.
6. Avoid unnecessary dependencies.
7. Prefer maintainable code over impressive code.

---

# Project Vision

The homepage is the product.

Visitors should immediately begin reading comics with minimal friction.

Navigation should remain lightweight and never distract from the stories.

---

# Design Language

The site should feel:

- Warm
- Friendly
- Personal
- Clean
- Modern
- Slightly playful
- Never corporate

Color palette:

- White backgrounds
- Black typography
- Señor Rosa pink as the primary accent

Whitespace is preferred over visual clutter.

---

# Comic Philosophy

Stories are based on real experiences.

Humor should come from:

- cultural misunderstandings
- language mistakes
- travel
- everyday life
- curiosity
- self-deprecation

Never mock people or cultures.

The joke is usually on Nico.

---

# AI Expectations

When generating code:

- Keep files small.
- Explain architectural changes.
- Avoid unnecessary frameworks.
- Do not rewrite working code without reason.
- Preserve existing styling unless requested.

When proposing features, ask:

"Does this help tell stories?"

If the answer is no, don't build it.

---

# Preferred Architecture

Simple static website.

Preferred technologies:

- HTML
- CSS
- JavaScript

Only introduce additional tools when they clearly reduce long-term complexity.

---

# Publishing Workflow

Memory

↓

Comic script

↓

Artwork

↓

Website

↓

Publish

The publishing pipeline is the primary workflow.

Optimize for making the next comic easier to publish.

---

# Long-Term Vision

Over time the site may include:

- comics
- travel journals
- consulting portfolio
- biography
- AI experiments
- Spanish learning
- photography

However, comics remain the centerpiece.

Every new feature should support the Señor Rosa brand rather than compete with it.
